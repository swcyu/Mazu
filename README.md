# 전시 추천 서비스_MAZU

## 개요
- Linux 서버 운영 및 사용자 로그 수집 프로젝트
- 기술 / 환경
  - Python
  - Elasticsearch
  - Logstash
  - Kibana
  - Liunx (Ubuntu 18.04)
- 기간: 2022.10.11 ~ 2022.10.28 (14일)

## 내용
- AWS 기반으로 구현된 전시 추천 서비스를 로컬 Linux 서버 기반으로 전환하는 작업을 진행하였습니다.
- 기존의 서비스에 ELK 스택을 덧붙여 사용자 로그를 수집하여 이를 바탕으로 전시를 추천하였습니다.
